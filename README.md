# Zepto Sales Performance Dashboard

## 📊 Project Overview
This dashboard analyzes sales performance, customer behavior, and order trends for Zepto.

## 🔧 Tools Used
- Power BI
- Excel

## 📌 Key Insights
- Total sales reached 2M with 1500 orders
- UPI is the most preferred payment method
- Chennai has the highest number of orders
- Delivery success rate is around 60%
- Sales trend increases towards year-end
